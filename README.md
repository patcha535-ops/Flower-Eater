# 🌸 Flower Eater - Jarvis Final Edition

**Flower Eater** เป็นเกมแนว Sliding Puzzle ขนาด 7x7 ที่พัฒนาด้วย HTML5, CSS3 และ Vanilla JavaScript (ES6+) โดยมีกลไกการเล่นที่เป็นเอกลักษณ์เฉพาะตัว

## 🎮 กลไกการเล่นแบบใหม่ (Game Mechanics)

1. **การเริ่มต้น (The First Gap)**: เมื่อเริ่มเกม ดอกทิวลิป (🌷) จะถูกล็อคตำแหน่งไว้ที่ **ใจกลางกระดาน (พิกัด 3,3)** เสมอ และเมื่อผู้เล่นคลิกที่ดอกไม้ดอกแรก ระบบจะทำการ **สุ่มทำลายดอกไม้ 1 ดอก** เพื่อสร้างช่องว่างเริ่มต้นให้
2. **การเคลื่อนที่**: ผู้เล่นสามารถคลิกดอกไม้ที่อยู่ติดกับช่องว่างเพื่อดัน (Push) ดอกไม้เหล่านั้นให้เลื่อนไปในทิศทางที่ต้องการ
3. **ระบบตัวกินดอกไม้ (Eater Logic)**: ตัวกินที่มุมทั้ง 3 จะทำงานก็ต่อเมื่อดอกไม้ที่มีชนิดตรงกันถูกดันไปวางที่พิกัดมุมนั้นพอดี:
   - 🌹 **มุมบนซ้าย (0,0)**: กินเฉพาะดอกกุหลาบ
   - 🌻 **มุมบนขวา (0,6)**: กินเฉพาะดอกทานตะวัน
   - 🌷 **มุมขวาล่าง (6,6)**: กินเฉพาะดอกทิวลิป (เป้าหมายหลักเพื่อจบเกม)

## 🛠️ ข้อมูลทางเทคนิค

- **Grid System**: 7x7 Responsive Grid
- **Strict Matching**: ระบบตรวจสอบพิกัดและชนิดดอกไม้แบบเจาะจง ป้องกันการกินผิดช่อง
- **State Management**: ใช้สถานะ `gameStarted` เพื่อควบคุมจังหวะการสุ่มช่องว่างครั้งแรก
